# Divide and Conquer Algorithms – Assignment 2

## Overview
This repository contains the analysis and implementation of two divide-and-conquer algorithms:

- Merge Sort
- Quick Sort

The assignment includes theoretical analysis using asymptotic notation, recurrence relations, and practical implementation in Python.

## Algorithms Implemented

### Merge Sort
- Divide the list into two halves
- Recursively sort each half
- Merge the sorted halves

### Quick Sort
- Select a pivot element
- Partition the array
- Recursively sort the partitions

## Performance Testing

The algorithms were tested using datasets of **5000 elements**:

- Random data
- Sorted data
- Reverse sorted data

Metrics recorded:
- Execution time
- Memory usage
